Como funciona o HTML

O HTML, ou HyperText Markup Language, é uma linguagem de marcação usada para criar a estrutura de páginas na internet.

A estrutura de uma página HTML é interpretada, processada e exibida pelos navegadores, conforme as tags configuradas no código.

Os documentos HTML são arquivos que possuem extensão .html ou .htm.


Estrutura básica do HTML

Todo documento HTML começa com uma estrutura básica, que é praticamente padrão e sempre deve ser seguida para garantir o funcionamento correto da página.

Editores de código como Sublime Text ou Visual Studio Code possuem atalhos que facilitam a criação dessa estrutura inicial.


A tag <html> é a raiz do documento e indica ao navegador que o conteúdo é código HTML, permitindo que ele seja interpretado corretamente.

A tag <head> contém informações não visíveis diretamente ao usuário, como vínculos com arquivos externos, scripts e metadados.

A tag <body> representa o corpo da página, onde está todo o conteúdo visível ao usuário, incluindo textos, imagens, vídeos e formulários.


Falando sobre tags

Tags são os elementos básicos que definem a estrutura e o significado do conteúdo em uma página web.

Elas atuam como etiquetas que indicam o início e o fim de diferentes tipos de conteúdo — como títulos, parágrafos, imagens e links — ajudando a organizar e apresentar o conteúdo visualmente.


As tags são escritas entre os símbolos de "menor que" e "maior que" (< e >), por exemplo: <p>, <h1>, <img>.

Geralmente, as tags aparecem em pares: uma de abertura e outra de fechamento. A tag de fechamento contém uma barra (/) antes do nome da tag, por exemplo: <p>...</p>.

Algumas tags são auto-fecháveis (self-closing), ou seja, não precisam de uma tag de fechamento separada, como <br/> (quebra de linha) e <img/> (imagem).

Tags Textuais

Estruturais (bloco de texto) — definem hierarquia e organização:

  • <h1>, <h2>, <h3>…: cabeçalhos usados para títulos e subtítulos, com hierarquia decrescente de importância.
  • <p>: define um parágrafo, usado para textos corridos.
  • <blockquote>: representa uma citação destacada.

Formatação (semânticas e não semânticas) — definem aparência e significado:

  • <strong>: ênfase forte, texto em negrito com significado semântico.
  • <b>: texto em negrito, sem ênfase semântica.
  • <u>: texto sublinhado.
  • <i>: texto em itálico, sem ênfase semântica.
  • <mark>: texto marcado ou destacado.
  • <sup>: sobrescrito (exemplo: x²).
  • <sub>: subscrito (exemplo: H₂O).

Tags de listas

  • <ol>: lista ordenada (com números ou letras).
  • <ul>: lista não ordenada, com marcadores.
  • <li>: item da lista, usado dentro de <ol> ou <ul>.

Tags de links

Links conectam uma página a outra ou a uma seção específica.

A tag <a>, conhecida como âncora, cria links para outras páginas, seções do mesmo documento, arquivos ou ações (como enviar e-mail). O destino do link é definido pelo atributo href. 

<a href="https://www.example.com">Visitar site</a>

Atributos das tags

Atributos são palavras-chave usadas dentro de tags para fornecer informações adicionais ou modificar seu comportamento. Cada atributo tem um nome e um valor. Exemplos comuns:

  • id: identificador único para referência, geralmente usado em JavaScript.
  • class: especifica um ou mais nomes de classe para estilização ou lógica.
  • style: define estilos CSS diretamente no elemento.
  • src: especifica o caminho para um arquivo, como imagens ou scripts.
  • type: define o tipo de dado em campos de entrada (inputs).
